# Apache Ranger 组件介绍

## 概述

Apache Ranger 提供集中化的安全管理框架，用于在 Hadoop 生态系统中管理细粒度的数据访问控制、审计和策略管理。它通过轻量级 Plugin 嵌入到各大数据组件中，在数据访问点实时执行安全策略。

## 核心特性

- **统一策略管理**：通过 Web UI 集中管理所有组件的访问策略
- **细粒度访问控制**：支持数据库、表、列、行级别的权限控制
- **标签策略（Tag-Based）**：结合 Atlas 标签实现跨组件的数据安全
- **行过滤与列脱敏**：支持动态数据脱敏和行级安全
- **集中审计**：所有访问操作记录可查，支持合规审计

## 系统架构

- **Ranger Admin**：Web 管理界面，定义和管理安全策略
- **Ranger Plugin**：嵌入到 HDFS、Hive、HBase、Kafka 等组件中的轻量级 Java 插件
- **Ranger UserSync**：从 LDAP/AD 同步用户和组信息
- **Audit**：集中收集和处理访问日志
