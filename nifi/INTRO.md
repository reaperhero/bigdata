# Apache NiFi 组件介绍

## 概述

Apache NiFi 是一个数据流自动化平台，提供基于 Web 的直观拖拽界面，用于设计、控制和监控数据在系统之间的流转。它最初由 NSA 开发（项目代号 Niagara Files），后贡献给 Apache 基金会。

## 核心特性

- **可视化界面**：拖拽式流程设计，实时监控数据流状态
- **300+ 内置处理器**：支持数据库、文件系统、Kafka、HDFS、HTTP 等多种数据源
- **数据溯源（Data Provenance）**：跟踪每条数据的完整流转历史
- **动态调整**：运行中可实时修改数据流，无需停止服务
- **背压和优先级**：内置队列管理，防止数据积压导致系统过载

## 核心概念

| 概念 | 说明 |
|------|------|
| FlowFile | 数据的"信封"，包含数据内容和属性元数据 |
| Processor | 执行具体任务的模块化组件（获取、转换、路由、发送） |
| Connection | 连接 Processor 的队列，FlowFile 在其中排队等待处理 |
| Process Group | 将相关 Processor 组织在一起的逻辑分组 |
